TWITTER CLONE

[ CLIENT - Front-End ]
> HTML / CSS
> Skeleton.css

[ SERVER - Back-End ]
> Static File Server (Live-Server)
> Dynamic Server (JS, Node.js)- JSON API
> Database - MongoDB (Persistant Storage)

> NPM, NODEjs
> Express
> Morgan (Middleware)
> nodemon (automatically refresh when updated) - Dev Tool - [ npm i --save-dev nodemon ]
> CORS (Express Middleware) - [ npm i cors]
  - Adds 'Access-Control-Allow-Origin'
  - Install if you are in charge of server
> Monk (MongoDB API)


[ npm install bad-words ] - Filters derogatory words before adding to DB
[ npm install express-rate-limit ] - Limits to 1 request every 15s

mlab.com - Cloud-hosted MongoDB
zeit.co - Client/Server Deployment

[ START SERVER ]
- navigate to server folder
- cmd 'node index.js'

[ CLIENT FOLDER ]
- launch 'index.html' file
